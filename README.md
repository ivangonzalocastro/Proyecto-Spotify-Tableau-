# Proyecto-Spotify-Tableau
_Evolución de las tendencias musicales con Spotify_
## Descripción General
Este proyecto analiza más de 14.000 canciones de Spotify para entender cómo cambiaron sus características a lo largo de las décadas: popularidad, energía, acusticidad, ánimo, tonalidad y más.
El objetivo es ofrecer visualizaciones claras y dinámicas que permitan descubrir patrones musicales, entender tendencias y explorar cómo evolucionó el sonido moderno.

## Objetivo del Proyecto
Analizar y visualizar los principales parámetros de valoración que Spotify asigna a sus canciones, para:
* Identificar tendencias temporales.
* Comparar características musicales.
* Facilitar insights tanto para el público general como para músicos, productores o analistas curiosos por la evolución del sonido.

## Público Objetivo
* Personas interesadas en cómo Spotify analiza y clasifica la música.
* Músicos y productores (incluye una sección especial “Estadísticas para Músicos”).
* Analistas y fanáticos de los datos que quieran explorar tendencias musicales.
* Cualquier usuario curioso por la evolución de la música a lo largo del tiempo.

## Tecnologías Utilizadas
* Tableau
* Dataset de Kaggle
* Limpieza y preparación de datos realizada dentro de Tableau
  
## Skills aplicados en Tableau

Este proyecto incluye un uso intensivo de funcionalidades avanzadas de Tableau: 
* Creación de 23 campos calculados 
* Diseño de 5 parámetros interactivos
* Filtros dinámicos
* Acciones de navegación 
* Tooltip personalizados
* Visualizaciones variadas como gráficos de líneas, barras, donas, mapas de calor, tablas resaltadas y gráficos combinados.
* Técnicas de segmentación, normalización de métricas, agrupación por décadas, asignación de tonalidades musicales
* Control de visualización mediante cálculos IF, MAX y MIN para lograr gráficos limpios, consistentes y altamente interactivos.

## Hallazgos Destacados
* Las canciones más recientes tienden a tener mayor popularidad y aceptación, según los indicadores de Spotify.
* Las características musicales como energía, ánimo y acusticidad muestran cambios marcados entre décadas.
* La tonalidad y el modo musical también revelan variaciones interesantes vinculadas a géneros predominantes de cada época.
